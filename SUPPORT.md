# Support

Fulgurite community support is provided on a best-effort basis.

## Start Here

Before asking for help, check:

- [README.md](README.md)
- [Getting Started](docs/en/02-getting-started.md)
- [Concepts and Navigation](docs/en/03-concepts-and-navigation.md)
- [Settings Reference](docs/en/13-settings-reference.md)
- [ROADMAP.md](ROADMAP.md)

## Where To Ask

- Use public issues for reproducible bugs and actionable feature requests.
- Use pull requests for concrete fixes.
- Use private maintainer channels for security vulnerabilities, as described in [SECURITY.md](SECURITY.md).

## What To Include

Support requests are easier to handle when they include:

- the Fulgurite commit or version
- PHP version and extensions
- database engine
- web server
- operating system
- whether the issue affects setup, scheduler, worker, backup, copy, restore, API or notifications
- reproduction steps
- relevant logs with secrets redacted
- expected behavior and actual behavior

## Scope

Community support does not guarantee response time, deployment assistance, commercial use rights or private consulting.

Commercial support, managed hosting, paid features or commercial licensing require separate written terms when offered.
