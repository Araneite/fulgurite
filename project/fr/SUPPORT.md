# Support

Le support communautaire de Fulgurite est fourni au mieux des disponibilités.

## Commencer Ici

Avant de demander de l’aide, consultez :

- [README.md](README.md)
- [Installation et prise en main](../../docs/fr/02-getting-started.md)
- [Concepts et navigation](../../docs/fr/03-concepts-and-navigation.md)
- [Référence des paramètres](../../docs/fr/13-settings-reference.md)
- [ROADMAP.md](ROADMAP.md)

## Où Demander De L’aide

- Issues publiques pour les bugs reproductibles et demandes de fonctionnalités actionnables.
- Pull requests pour les corrections concrètes.
- Canaux privés mainteneur pour les vulnérabilités, selon [SECURITY.md](SECURITY.md).

## Informations Utiles

Incluez autant que possible :

- commit ou version Fulgurite
- version PHP et extensions
- moteur de base de données
- serveur web
- système d’exploitation
- zone concernée : setup, scheduler, worker, backup, copy, restore, API ou notifications
- étapes de reproduction
- logs sans secrets
- comportement attendu et comportement observé

## Périmètre

Le support communautaire ne garantit pas de délai de réponse, assistance de déploiement, droits commerciaux ou consulting privé.

Le support commercial, l’hébergement managé, les fonctionnalités payantes ou les licences commerciales nécessitent des conditions écrites séparées lorsqu’ils sont proposés.
