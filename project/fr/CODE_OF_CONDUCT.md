# Code De Conduite

Fulgurite doit rester un espace de collaboration calme, respectueux et techniquement sérieux.

## Comportements Attendus

Les participants doivent :

- rester respectueux et constructifs
- supposer la bonne intention avant d’escalader
- critiquer les idées, le code et les décisions sans attaquer les personnes
- accueillir les questions de personnes ayant différents niveaux d’expérience
- garder les discussions centrées sur le travail
- respecter les décisions mainteneur et les limites du projet

## Comportements Inacceptables

Ne sont pas acceptés :

- harcèlement, menaces ou intimidation
- propos discriminatoires ou haineux
- attaques personnelles, moqueries ou humiliations
- divulgation d’informations privées sans consentement
- perturbation volontaire, spam ou escalade de mauvaise foi
- publication d’informations de sécurité privées avant correction ou mitigation

## Responsabilités Des Mainteneurs

Les mainteneurs peuvent modérer les discussions, fermer des fils, retirer du contenu, refuser des contributions ou restreindre la participation si nécessaire pour protéger le projet.

## Signalement

Contactez les mainteneurs en privé lorsque possible.

Si l’incident concerne une vulnérabilité ou une information sensible, suivez [SECURITY.md](SECURITY.md).
